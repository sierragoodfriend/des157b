A list of features I plan to include

  • Point and click interactions – through the use of drawn outline png files and transparent shapes that can be hovered over.
  • A game in a new file with overlays and a health bar that decreases.
  • Pixelated backgrounds and characters.
  • AI voices.
  • Well researched infographics or informative text on hovers and in the game.
  • AI generated music.

The technologies required to build each feature

  • Photoshop to make png outlines.
  • Bing AI generator (or another AI) in order to create background images and characters.
  • The rest can be done in Visual Studio Code. I think my code is achievable without libraries – I wish I had used a library for the opacity revealing text animation in the window page, I might go back and try to do that if I have time, but I also like my solution!
  • An AI voice generator.
  • An AI music generator.
